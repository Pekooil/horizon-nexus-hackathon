# Ferret Casino Royal

## Description

You are a security gaurd at a casino and you need to catch all of the cheaters. You need to monitor all 6 rooms and when you find a cheater you need to catch them in a photo with your polaroid. If you survive all 5 nights, you win! But if you lose all of your money to the cheaters, you loose. To take a photo select a camera and make two Ls with your hands and connect them, thumb to index.

## Technologies

This game is made in Godot. OpenCV was used to detect if the player is making the take picture symbol with their hands.

## Deployment

This game is deployed on Vercel at <https://ferret-casino-camera-test.vercel.app/>
